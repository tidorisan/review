# ER図とは

* * *

db設計の手法の一つ。dbが必要なソフトウェアを開発する際は必ず用いる。様々な記法が存在しIE記法などが存在する。

# IE記法とは

* * *

関係[リレーション]と関連の多重度を示す。アイソレーションを表す記号。ER図でも使用されている。

| 記号    |    意味 |
| :---- | ----: |
| ○     | 0(ゼロ) |
| l     | 1(イチ) |
| （鳥の足） |     多 |
