| 指定             |  　フォームタイプ |
| :------------- | --------: |
| text_field     | テキストフィールド |
| email_field    |       メール |
| password_field |    パスワード　 |
| submit         |  サブミットボタン |

[注意]

データの型のみで自動的にfieldが指定されるわけではない

-   string
    -   text
    -   email
    -   password
